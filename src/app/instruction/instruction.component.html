How it works?

Create JSON file with this template:

{

"[Question 1]": {"type":"[Input Type]"},
"[Question 2]": {"type":"[Input Type]"}

}


Your types can be basic html types:

  • button
  • checkbox
  • color
  • date
  • datetime-local
  • email
  • file
  • hidden
  • image
  • month
  • number
  • password
  • range
  • reset
  • search
  • tel
  • text
  • time
  • url
  • week

Or custom type created by me:

  • select - in question in mustached bracket add options array like this:

    • {

    "[Question 1]": {"type":"[Input Type], "options":["[option 1]", "[option 2]", "[option 3]"]"},

    }
